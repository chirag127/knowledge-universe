# How to Solve It — Chapter-by-Chapter Summary

## Reading Guidance

This book has an unusual structure. Part I ("In the Classroom") is a Socratic dialogue on teaching problem solving. Part II ("How to Solve It") is a condensed dialogue showing the method in action. Part III ("Short Dictionary of Heuristic") is the main reference — 60+ alphabetized entries of heuristic strategies. Part IV provides problems with hints and solutions. Most readers find the dictionary (Part III) the most useful long-term resource.
