## 📝 Personal Notes

### Part 1: History
- Unique angle: traces 2,500-year tradition
- Simonides of Ceos (~500 BCE): origin of Method of Loci
- *Rhetorica ad Herennium* (~80 BCE): earliest surviving manual
- Cicero & Quintilian: memory as core of rhetoric
- Medieval shift: memory as meditation (Hugh of St. Victor)
- Renaissance: Memory Theatre (Camillo), occult memory (Bruno)
- Matteo Ricci: brought memory palace to China (1596)
- Gutenberg: printing press = decline of memory art
- Modern revival: World Memory Championships

- Peg System: number rhyme
- Face-Name Method: feature + keyword
- Keyword Method: sound-alike for languages
