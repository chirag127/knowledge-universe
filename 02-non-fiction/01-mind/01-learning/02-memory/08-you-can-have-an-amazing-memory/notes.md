**Author Bio:** Dominic O'Brien is an eight-time World Memory Champion, Grandmaster of Memory, and author of several books on memory training including *How to Develop a Brilliant Memory Week by Week*, *Learn to Remember*, and *How to Pass Exams*. He was diagnosed with dyslexia as a child and told he would not amount to much—making his memory championship success particularly remarkable.
