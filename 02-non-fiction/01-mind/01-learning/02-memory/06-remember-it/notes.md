## 📝 Personal Notes

### Chapter 1: My Story and Why You Should Listen to Me
- Grandmother with Alzheimer's as catalyst
- First competition: 16th place
- Inspired by Joshua Foer's *Moonwalking with Einstein*
- Core mission: practical, everyday memory tools

### Chapter 2: Basic Things You Must Remember Before You Start
- SEE-LINK-GO framework introduced
- Journey Method (Memory Palace) explained
- Number-Rhyme Peg: 1=bun, 2=shoe, 3=tree, 4=door, 5=hive, 6=sticks, 7=heaven, 8=gate, 9=vine, 10=hen
- Alphabet Peg: A=frame house, B=bee, C=sea, etc.
- Linking Method basics
- Importance of attention and observation

### Chapter 3: Brain Farts, Name-Amnesia...
- FACE: Focus, Ask, Connect, Endear
- Remembering where you parked: mental snapshot + verbal confirmation
- Keys: designated home + encoding specificity
- Directions: journey method applied to routes
- To-do lists: linking method on a short journey

### Chapter 4: For All the List-Based Things...
- Building effective journeys: 10-20 stops, familiar route
- Grocery lists: place items at journey stops
- US Presidents: journey with 45+ stops, each stop = one president
- Speeches: key points as images along a journey
- Reviewing journeys

### Chapter 5: Making Sense of the Gobbledygook Number Things...
- Number-Shape System: 0=ball, 1=pencil/candle, 2=swan, 3=heart/handcuffs, 4=sailboat/flag, 5=hook, 6=elephant trunk/golf club, 7=cliff/boomerang, 8=snowman/eyeglasses, 9=balloon/tennis racket
- Major System: 0=s/z, 1=t/d, 2=n, 3=m, 4=r, 5=l, 6=sh/j/ch/dg, 7=k/g/ck, 8=f/v, 9=p/b
- PAO System: 00-99, each = Person + Action + Object

### Chapter 6: Tips for Memorizing Some of the Other Things...
- Keyword Method for foreign language vocab
- Substitute words for unfamiliar terms
- Fact memorization: convert abstract to concrete images
- Active reading: summarize, visualize, place on journey

### Chapter 7: The Last Few Things...
- Spaced repetition schedule: 1 hr → 1 day → 1 wk → 1 mo → 1 yr
- Sleep: essential for memory consolidation
- Exercise: aerobic improves cognitive function
- Nutrition: brain-healthy diet
- Mindfulness: single-tasking over multitasking
- Curiosity: engage new topics regularly
- Practice: daily mental workouts
