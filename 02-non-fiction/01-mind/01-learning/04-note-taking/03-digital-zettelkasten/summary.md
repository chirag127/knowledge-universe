## 💡 The Big Idea in One Paragraph

A digital Zettelkasten is a bicycle for the mind — a low-friction system where fleeting notes become literature notes become atomic permanent notes, linked through specific keywords and file-naming conventions, turning your digital tool into an idea-generation engine that requires minimal maintenance and maximizes creative output.

## 🔑 Thematic Pillars & Core Frameworks

### 1. The Three Note Types (Classic Zettelkasten)
| Note Type | Description | Storage |
|-----------|-------------|---------|
| **Fleeting Notes** | On-the-fly captures; highlights, margin notes | Inbox → process |
| **Literature Notes** | Condensed summaries of sources in own words | Literature folder |
| **Permanent Notes** | Single atomic idea with metadata and links | Main slip-box |

### 2. Folder Structure
| Folder | Purpose |
|--------|---------|
| **Inbox** | Fleeting notes awaiting processing |
| **Someday/Maybe** | Interesting ideas not yet relevant |
| **Raw** | Exact exported highlights from sources |
| **Literature Notes** | Condensed source summaries |
| **Permanent Notes** | Atomic idea notes |

### 3. The Reading-to-Writing Workflow
1. Read and highlight (fleeting notes)
2. Export highlights → keep raw copies
3. Condense highlights into literature notes
4. Extract best ideas → permanent notes
5. Retrieve via keywords/links for writing

### 4. Keyword Strategy
Instead of generic tags (#psychology), use specific, unique keywords that reflect the insight itself (#icebergprinciple). Think like a writer, not an archivist.

### 5. Rituals & Positioning
Match mental state to activity:
- **Active/standing**: Exercise, brainstorm
- **Reclining**: Read, take fleeting notes
- **Sitting at computer**: Link, tag, organize
- **Phone (waiting)**: Review highlights, quick captures

## 🧠 Key Concepts Glossary

1. **Fleeting Notes** — Quick, rough captures destined for processing or deletion.
2. **Literature Notes** — Your-words summaries of books, articles, podcasts, conversations.
3. **Permanent Notes** — Single-idea atomic notes with keywords, links, and source metadata.
4. **Folgezettel** — German for "sequential note"; Luhmann's physical numbering system for ordering notes.
5. **Inbox** — Holding area for unprocessed fleeting notes.
6. **Raw Folder** — Storage for unedited source highlights/exports.
7. **Bicycle for the Mind** — Steve Jobs' metaphor for computers; Kadavy applies it to the Zettelkasten.
8. **Iceberg Principle (keyword example)** — Specific insight-based tag instead of generic category.
9. **Contextual Phrase** — Alternative title line for improved searchability.
10. **Active vs. Passive Note-Making** — Rewriting (active) vs. copying/pasting (passive).
11. **Low-Effort Production** — The Zettelkasten allows productive work in short time windows.
12. **Metabolic State Matching** — Doing the right type of note work for your current energy level.
