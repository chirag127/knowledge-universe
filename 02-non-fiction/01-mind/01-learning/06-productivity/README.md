# Productivity & Focus

Welcome to the **Productivity & Focus** section of your Library of Alexandria. This branch covers task management systems, time prioritization models, defeating distraction, habit building, and techniques for cultivating high-level focus and execution.

---

## 🧠 Core Principles of Productivity & Focus

1. **System-Based Capture**: Reducing cognitive load on working memory by writing down and organizing tasks in an external, reliable system (e.g., GTD).
2. **Atomic Priorities**: Identifying and concentrating resources on the single highest-value action item (e.g., The ONE Thing) while saying no to the trivial many.
3. **Deep Work Scheduling**: Dedicating distraction-free blocks of time to cognitively demanding activities to produce high-value output.
4. **Behavioral Habit Loops**: Designing daily systems and cues to make execution automatic and friction-free (e.g., Atomic Habits).
