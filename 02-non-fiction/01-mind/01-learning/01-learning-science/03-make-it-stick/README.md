# Make It Stick: The Science of Successful Learning

## Metadata
* **Authors**: Peter C. Brown, Henry L. Roediger III, Mark A. McDaniel
* **Published**: 2014
* **Description**: Explains how cognitive psychology's empirical findings prove that memory retrieval, spaced practice, and interleaving are crucial for durable learning, debunking common study myths like rereading and highlighting.
