# Book Summary: Learning and Memory: From Brain to Behavior

## Overview
This textbook provides an integrated, neuroscientific view of memory systems. It contrasts behavioral conditioning (classical and operant) with cognitive structures. It explores the physical neuroanatomy of episodic and semantic memory in the hippocampus and cortex, investigates skill memory and motor learning in the basal ganglia and cerebellum, dissects working memory and cognitive control in the prefrontal cortex, and covers social learning, emotional memory, and age-related cognitive development.

---

### Core Learning Takeaways
* **Pillar 1**: The premier collegiate textbook bridging cognitive psychology, behavioral theory, and neurobiology to explain the neural and mechanical structures behind how humans learn and remember.
* **Pillar 2**: Real-world application of memory, consolidation, and practice.
* **Pillar 3**: Overcoming study illusions through active, effortful retrieval.
